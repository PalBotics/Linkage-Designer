# Linkage Designer MVP Spec

* Vite + React + TypeScript + Zustand web app rendering mechanisms as SVG.
* Kinematics package providing geometry helpers, joint solver, and type definitions.
* JSON example `examples/viseGripPipeWrench.json` demonstrating a simple mechanism.
* I/O utilities for JSON loading/saving and CSV/SVG/PNG exports.
* `pnpm -w dev` starts the web app; `pnpm -w test` runs package tests.
* GitHub Actions workflow builds and tests on push.
